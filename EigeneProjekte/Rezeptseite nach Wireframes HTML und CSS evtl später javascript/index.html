Lasagne

Lasagne hat eine lange Geschichte, die bis ins Mittelalter zurückreicht. Ein Rezept für ein Gericht, das der heutigen Lasagne ähnelt, wurde im 14. Jahrhundert in einem mittelalterlichen Kochbuch namens "Liber de Coquina" gefunden.

Braten

Der erste Braten, im Sinne von Fleisch, das über offenem Feuer oder in heißer Asche gegart wurde, ist sehr alt und geht auf die Zeit zurück, als der Mensch das Feuer entdeckte und nutzte. Erste Hinweise auf das Garen von Fleisch über Feuer gibt es bereits vor etwa 1,5 Millionen Jahren mit dem Homo erectus.
