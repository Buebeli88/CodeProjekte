CSS Box Model

Das CSS-Boxmodell beschreibt, wie Browser Elemente auf einer Webseite darstellen, indem sie jedes Element als rechteckige Box behandeln. Diese Box besteht aus vier Teilen: Inhalt, Innenabstand (padding), Rahmen (border) und Außenabstand (margin). Jedes dieser Elemente trägt zur Größe und Positionierung der Box bei.

Box-Sizing: border-box

Der Wert border-box der CSS-Eigenschaft box-sizing entspricht direkt der insgesamt gerenderten Größe eines Elements – einschließlich Padding (Innenabstand) und Border (Rahmen) – in Verbindung mit den Höhe- und Breitenangaben. Der Standardwert der Eigenschaft box-sizing ist content-box. Der Wert border-box wird empfohlen, wenn es notwendig ist, Padding und Rahmen mit anzupassen, nicht aber nur den Inhalt. Zum Beispiel berechnet der Wert border-box die Höhe eines Elements wie folgt: Höhe = Inhaltshöhe + Padding + Rahmen.

Beispiel:

.container{
  box-sizing: border-box;
}

Höhe und Breite - Min/Max

Die CSS-Eigenschaften min-width und min-height können verwendet werden, um eine minimale Breite und minimale Höhe eines Element-Containers festzulegen. Die Eigenschaften max-width und max-height können verwendet werden, um maximale Breiten und Höhen für Element-Boxen zu definieren.

Beispiel:

(Trotz des Breitenwerts 500px wird diese Box maximal 200px breit sein.)

.column{
  max-width: 200px;
  width: 500px;
}

Margin Collapse

CSS Margin Collapse (Zusammenfallen von Rändern) tritt auf, wenn die oberen und unteren Außenabstände (Margins) von Blockelementen zu einem einzigen Abstand zusammengefasst werden, der dem größten einzelnen Margin-Wert entspricht. Margin Collapse tritt nur bei vertikalen Rändern auf, nicht bei horizontalen.

Beispiel:

(Statt einem Abstand von 50px, entsteht hier ein Abstand von 30px)

.block-one{
  margin: 20px;
⤍

.block-two{
  margin: 30px;
}

auto Keyword

Der Wert auto kann zusammen mit der CSS-Eigenschaft margin verwendet werden, um ein Element horizontal innerhalb seines Containers zu zentrieren. Die margin-Eigenschaft berücksichtigt dabei die Breite des Elements und verteilt den restlichen verfügbaren Platz gleichmäßig auf den linken und rechten Außenabstand.

Beispiel:

Div{
  margin: auto;
}

Dealing with overflow

Wenn der Inhalt zu groß für seinen Container ist, legt die CSS-Eigenschaft overflow fest, wie der Browser mit diesem Problem umgeht. Standardmäßig ist der Wert auf visible gesetzt, wodurch der Inhalt zusätzlichen Platz einnimmt. Alternativ kann overflow auf hidden gesetzt werden, um den überlaufenden Inhalt auszublenden, oder auf scroll, wodurch der überlaufende Inhalt über Scrollleisten innerhalb des ursprünglichen Containers zugänglich gemacht wird.

Beispiel:

.small-block{
  overflow: scroll;
}

The visibility Property

Die CSS-Eigenschaft visibility wird verwendet, um versteckte Elemente für den Benutzer unsichtbar zu machen, ohne sie von der Seite zu entfernen. Dadurch bleibt die Struktur und Organisation der Seite unverändert.

Beispiel:

.invisible-elements{
  visibility: hidden;
}
