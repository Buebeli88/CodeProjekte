Display and Positioning

In CSS bezeichnen display und position zwei fundamentale Eigenschaften zur Gestaltung des Layouts von Webseiten. display bestimmt, wie ein Element als Box gerendert wird (z.B. als Block, Inline-Element oder als Teil eines Flex- oder Grid-Layouts). position legt fest, wie ein Element im Dokument positioniert wird, also wie es sich zu anderen Elementen oder dem Viewport verhält.

Positionierung in CSS bietet Designern und Entwicklern Möglichkeiten, HTML-Elemente auf einer Webseite zu positionieren. Die CSS-Eigenschaft position kann auf static, relative, absolute oder fixed gesetzt werden.

"display" Property

Die CSS-Eigenschaft display bestimmt die Art der Darstellung eines Elements. Die gebräuchlichsten Werte für diese Eigenschaft sind block, inline und inline-block.

Block-Level-Elemente nehmen die volle Breite ihres Containers ein, erzeugen Zeilenumbrüche davor und danach und können in Höhe und Breite manuell angepasst werden.

Inline-Elemente nehmen so wenig Platz wie möglich ein, fließen horizontal und können in Höhe und Breite nicht manuell angepasst werden.

Inline-Block-Elemente können nebeneinander angezeigt werden und ihre Höhe sowie Breite können manuell angepasst werden.

Beispiel:

.container1{
  display: block;
}

.container2{
  display: inline;
}

.container3{
  display: inline-block;
}

Fixed CSS Positioning

Hat die CSS-Position den Wert fixed, wird das Element an einer festen Stelle auf der Seite verankert. Das feste Element bleibt unabhängig vom Scrollen immer an derselben Position.

Die Navigationsleiste ist ein gutes Beispiel für ein Element, das häufig auf position: fixed; gesetzt wird, sodass der Benutzer durch die Webseite scrollen kann und dennoch jederzeit Zugriff auf die Navigationsleiste hat.

Beispiel:

#navbar{
  position: fixed;
}

Postition: relative

Der Wert relative der CSS-Eigenschaft position ermöglicht es, ein Element relativ zu seiner ursprünglichen Position auf einer Webseite zu positionieren. Die Offset-Eigenschaften können verwendet werden, um die tatsächliche Position des Elements relativ zu seiner ursprünglichen Position festzulegen. Ohne die Offset-Eigenschaften hat diese Deklaration keine Auswirkung auf die Positionierung und verhält sich wie der Standardwert static der Eigenschaft position.

Beispiel:

.element{
  position: absolute;
}

Position: absolute

Der Wert absolute der CSS-Eigenschaft position ermöglicht es einem Element, Geschwisterelemente zu ignorieren und stattdessen relativ zu seinem nächstgelegenen Elternelement positioniert zu werden, das entweder relativ oder absolut positioniert ist. Der Wert absolute entfernt ein Element vollständig aus dem Dokumentenfluss. Durch die Verwendung der Positionsattribute top, left, bottom und right kann ein Element wie erwartet an jeder beliebigen Stelle positioniert werden.

Beispiel:

.element{
  position: absolute;
}

Position: fixed

In CSS bedeutet position: fixed, dass ein Element relativ zum Browserfenster (Viewport) positioniert wird und immer an derselben Stelle bleibt, auch wenn die Seite gescrollt wird. Das Element wird also nicht mit dem Inhalt verschoben, sondern bleibt fest an seiner Position verankert.

Beispiel:

.element{
  position: fixed;
}

Position: sticky

"Position: sticky" in CSS ist eine Positionierungsmethode, die ein Element zwischen relativer und fixer Positionierung umschalten lässt, basierend auf der Scrollposition des Benutzers. Es verhält sich zunächst wie ein Element mit position: relative, bis es einen bestimmten Schwellenwert im Viewport erreicht (z.B. 10px von oben), und wechselt dann zu position: fixed und bleibt an dieser Stelle "kleben", während der Rest der Seite weiter scrollt.

Beispiel:

.element{
  positin: sticky;
}

Property: float

Die CSS-Eigenschaft float bestimmt, wie weit links oder rechts ein Element innerhalb seines Elternelements schweben soll. Der Wert left lässt ein Element an der linken Seite seines Containers schweben, und der Wert right lässt ein Element an der rechten Seite seines Containers schweben. Für die Eigenschaft float muss die Breite des Containers angegeben werden, ansonsten nimmt das Element die volle Breite seines enthaltenden Elements ein.

Beispiel:

.left{
  float: left;
}

.right{
  float: right;
}

Property: clear

Die CSS-Eigenschaft clear legt fest, wie sich ein Element verhalten soll, wenn es innerhalb desselben übergeordneten Elements auf ein anderes Element stößt. clear wird üblicherweise in Kombination mit Elementen verwendet, die die CSS-Eigenschaft float besitzen. Damit wird bestimmt, auf welchen Seiten schwebende (floatende) Elemente erlaubt sind zu schweben.

Beispiel:

.element{
  clear: left;
}

.element{
  clear: right;
}

.element{
  clear: both;
}

.element{
  clear: none;
}

Property: z-index

Die CSS-Eigenschaft z-index legt fest, wie weit hinten oder wie weit vorne ein Element auf einer Webseite erscheint, wenn es sich mit anderen Elementen überlappt. Die Eigenschaft z-index verwendet ganzzahlige Werte, die positiv oder negativ sein können. Das Element mit dem höchsten z-index-Wert befindet sich im Vordergrund, während das Element mit dem niedrigsten z-index-Wert im Hintergrund liegt.

Beispiel:

.element1{
  position: absolute;
  z-index: 1;
}

.element2{
  position: absolute;
  z-index: -1;
}

Positioning: Footer

Es gibt verschiedene Angehensweisen den Footer immer unten zu positionieren. Eine der einfacheren Methoden, ist "old school". In welcher man den Inhalt des "Bodys" in einen "div" einpackt.

Beispiel:

HTML
<body>
  <div class="container">
    <h1>Beispiel</h1>
    <p>Beispiel Text</p>
    etc...
  </div>
  <footer>DER FUSS</footer>
</body>

Danach gibt man dem "container" noch eine min.höhe vom "Viewport" minus der Footer Höher.

Beispiel:

.container{
  min-height: calc(100vh - 55px);
}

Border Radius

In CSS bezeichnet border-radius die Eigenschaft, mit der die Ecken eines HTML-Elements abgerundet werden können. Sie ermöglicht es, die Ecken von Elementen wie Rechtecken oder Quadraten mit einem Radius zu versehen, wodurch sie eine runde oder elliptische Form erhalten.

Beispiel:

.navigation{
  border-radius: 5px;
}

oder

.navigation{
  border-radius: 10%;
}
