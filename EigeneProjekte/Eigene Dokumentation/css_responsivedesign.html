Responsive Design

Responsive Design in CSS ist ein Ansatz zur Gestaltung von Webseiten, der sicherstellt, dass die Webseite auf verschiedenen Bildschirmgrößen und Geräten optimal dargestellt wird. Es passt das Layout, die Bilder und den Text automatisch an die Bildschirmgröße des Geräts an, das der Benutzer verwendet, sei es ein Desktop-Computer, ein Tablet oder ein Smartphone.

Media Features in Media Queries

Media Queries in der Webentwicklung stellen sicher, dass ein Website-Design responsiv ist. Dies geschieht durch die Erstellung maßgeschneiderter Stylesheets, die auf der Auflösung des Geräts basieren, auf dem die Website angezeigt wird. Der Browser erkennt die Bildschirmgröße und wendet die CSS-Stile für diese Bildschirmgröße basierend auf den angegebenen Media-Features an. Media Queries können Regeln festlegen, die sich auf Eigenschaften wie Bildschirmbreite, Bildschirmauflösung und Geräteausrichtung beziehen. Diese Media-Regeln können durch Kommata getrennt werden. Wenn eine der Media-Regeln zutrifft, wird das zugehörige CSS angewendet.

Ranges in Media Queries

Media Queries können CSS verwenden, um Bildschirmgrößen innerhalb eines bestimmten Bereichs anzusprechen, indem mehrere Breiten und/oder Höhen verwendet werden. Dies ist ein effektives Werkzeug für responsives Design, das eine Vielzahl von Bildschirmgrößen mit einer einzigen CSS-Media-Query abdecken kann. Um einen Bereich für die Breite oder Höhe festzulegen, wird die minimale Bildschirmgröße mit min-width und/oder min-height definiert und die maximale mit max-width oder max-height. Diese Eigenschaften werden in Kombination mit dem and-Operator verwendet.

Beispiel:

@media only screen and (min-width: 480px) and (max-width: 600px){
  (ruleset for 480px - 600px)
}

Media Query

Eine CSS Media Query kann verwendet werden, um die Darstellung und das Layout einer Website an verschiedene Bildschirmgrößen anzupassen. Eine Media Query beginnt mit dem @media-Schlüsselwort und wird von einer oder mehreren Bedingungen gefolgt, die Bildschirmgröße, Ausrichtung, Auflösung und/oder andere Eigenschaften prüfen. Wenn diese Bedingungen erfüllt sind, werden alle CSS-Regeln innerhalb der Media Query auf die Seite angewendet. Media Queries werden im responsiven Webdesign verwendet, um durch gezielte Stylesheets verschiedene Gerätetypen wie Laptops, Tablets, Mobilgeräte und mehr anzusprechen.

Beispiel:

Für Bildschirmgrößen kleiner oder gleich 480px (höchstwahrscheinlich ein mobiles Gerät) wird die Schriftgröße des <body>-Elements auf 12px gesetzt und die Breite des Elements mit der ID #photo auf 100%

@media only screen and (max-width: 480px){
  body{
    font-size: 12px;
  }

  #photo{
    width: 100%;
  }
}

Media Type in Media Queries

Beim Schreiben von Media Queries in CSS verwenden wir das Schlüsselwort only, um den Gerätetyp anzugeben. Ursprünglich unterstützte CSS eine Vielzahl von Medientypen wie screen, print und handheld. Um responsives Design zu gewährleisten und verschiedene Bildschirmgrößen zu berücksichtigen, wird heutzutage das Schlüsselwort screen immer für die Anzeige von Inhalten verwendet.

Beispiel:

@meida only screen and (min-width: 600px){
  Ruleset for >=600px
}

Operator and in Media Queries

Durch die Verwendung des and-Operators beim Schreiben von Media Queries in CSS können wir angeben, dass mehrere Media-Features erforderlich sind. Zum Beispiel könnten wir eine bestimmte Breite sowie eine bestimmte Auflösung verlangen, damit ein CSS-Regelsatz angewendet wird. Der and-Operator erlaubt es, mehrere Media-Features miteinander zu verknüpfen, wodurch wir gezielter auf bestimmte Bildschirmtypen eingehen können, während wir responsive Designs erstellen.

Beispiel:

@media only screen and (max-width: 480) and (min-resolution: 300dpi){
  Ruleset
}

Benutzung von Prozenten in CSS

Anstatt eine feste Breite mit Einheiten wie px oder cm zu definieren, können in CSS Prozentsätze verwendet werden, um die Höhe und Breite von Kindelementen relativ zu den Abmessungen ihres Elternelements festzulegen. Die Kindelemente wachsen oder schrumpfen entsprechend den angegebenen Prozentwerten, wenn sich die Größe des Elternelements ändert.

Beispiel:

.news-row{
  height: 300px;
  width: 500px;
}

.news-row .news-column{
  height: 80%; (240px)
  width: 50%; (250px)
}
