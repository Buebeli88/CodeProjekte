CSS Typography

Unter CSS Typography versteht man die Anwendung von Cascading Style Sheets (CSS) zur Gestaltung der Typografie, also der Schriftart, Schriftgröße, Schriftstil, Schriftstärke und des Textlayouts auf Webseiten.

Font Family

Die "Property" font-family wird benutzt um die Schriftart zu setzen. Der Browser muss in der Lage sein den gewählten "Font" auch darzustellen. Der "Font" sollte demnach entweder auf dem Gerät vorhanden sein oder als "web-font" gelinkt werden. Hat ein "Font" Name mehrere Wörter, werden diese in "" Gänsefüschen gehalten.

Beispiel:

Wie man die generelle Schriftart des Dokuments/Browsers setzt.

html {
font-family: Arial;
}

Oder ein Spezifisches Element (Welches die generelle Schriftart überschrieben wird).

h2 {
font-family: "Times New Roman";
}

Here's a Text in "Times New Roman"

oder

.header {
font-family: "Franklin Gothic Medium";

Here's a Text in "Franklin Gothic Medium

CSS @font-face Rule

Die @font-face Rule ermöglicht das importieren von "external" Fonts direkt ins Stylesheet. Die "Ablage" muss spezifisch in dieser "Rule" angegeben werden. Sie ermöglicht auf das laden von Lokalen Fonts.

Beispiel:

@font-face {
font-family: Gleegoo;
src: url ("../resources/fonts/Gleegoo-Regular.ttf") oder den web-link zu diesem "Font"
format ("truetype");

CSS Linking Fonts

Durch "Linking Fonts" ist es möglich der Seite einen WebFont für dein Dokument zu geben. Diese Fonts werden durch einen <link> Tag im HTML Dokument im <head> eingefügt. Soblad die URL im "href" Attribut ist, kann dieser Font benutzt werden.

Beispiel:

<head>
  <link href="https://fonts.googleapis.com/css?family=Droid%20Serif" rel="stylesheet">
</head>

CSS Fallback Fonts

Es ist sinnvoll beim setzen einer Font-Family auch "Fallback Fonts" zu setzen. Diese werden benutzt, wenn der "Haupt-Font" vom Browser nicht geladen werden kann. In der Font-Family sind alle Fonts hinter dem ersten ein "Fallback Font". Zudem sollte als letzer "Fallback Font" ein "serif" oder "sans-serif" gesetzt werden.

Beispiel:

p {
font-family: Helvetica, Arial, serif;
}

Font-Size

Die "Font Size" gibt an wie gross der ausgewählte Text dargestellt werden soll. Dafür können verschiedene Einheigen benutzt werden.

Übliche Einheiten:
px, em und rem

Beispiel:

p {
font-size: 1rem;
}

Unit em

In CSS wird die Einheit em benutzt um die Relative grösse des Kontents darzustellen. 1 em Einheit representiert die "Base Font Size" des aktuellen Elements. Im Beispielcode wird die Textgrösse im <span> Element 15px gross sein.

Beispiel:

.nav-container {
font-size: 10px;
}

.nav-bontainer span {
font-size: 1.5em; (10 x 1.5 = 15px)
}

Unit rem

Die Einheit rem wird benutzt umd die Grösse im Verhältnis zum "Root"Element zu setzen, welche im <html> gesetzt wird.

Beispiel:

html {
font-size: 18px;
}

span {
font-size: 2rem; (18 x 2 = 36px)
}

font-weight Property

Das "Font Weight" bestimmt wie dick oder dünn ein Text dargestellt werden soll. Dies kann durch Zahlen von 100 bis 900 in 100er Schritten dargestellt werden, Oder auch durch Text wie "Bold", "Bolder". Bold entspricht einem Wert von 400 und Bolder von 700.

Beispiel:

p{
font-weight: 700;
}

oder

p{
font-weight: bolder;
}

Dieser "Text" wird dicker.

Text Align

Dies wird genutzt um die Ausrichtung des Textes zu bestimmen. Es kann left, right oder center ausgewählt werden.

Beispiel:

p{
text-alignt: right;
}

font-style Property

Dies entscheidet den Style des gewählten Textes und kann auf "italic" gesetzt sein. (Gleich die <em> in HTML)

Beispiel:

p{
font-style: italic;
}

line-height Property

Die "line-heigt" gibt an, wieviel Platz zwischen den Textlinien dargestellt wird. Es können Zahlen mit oder ohne Einheiten verwendet werden. Die benutzten Einheiten sind px, %, em und rem. Der angegebene Wert kann nie negativ sein.

Beispiel:

p{
line-height: 10px;
}
